# 小米澎湃OS样式改造计划

## 1. 设计系统改造

### 1.1 色彩系统
- 主色调：小米橙色 (#FF6A00)
- 辅助色：浅橙色 (#FFEBE0)
- 背景色：白色 (#FFFFFF)、浅灰色 (#F5F5F5)
- 文本色：深灰色 (#333333)、中灰色 (#666666)、浅灰色 (#999999)
- 边框色：极浅灰色 (#E5E5E5)

### 1.2 排版系统
- 字体：小米兰亭Pro或系统默认无衬线字体
- 字体大小：
  - 标题：24px、20px、18px
  - 正文：16px、14px
  - 辅助文字：12px
- 行高：1.5-1.8

### 1.3 圆角和阴影
- 圆角：8px、12px、16px
- 阴影：轻微阴影效果，增强层次感

## 2. 组件样式改造

### 2.1 Layout组件
- Header：白色背景，底部1px边框，高度60px
- Content：白色背景，适当内边距
- Footer：浅灰色背景，居中文字

### 2.2 Menu组件
- 水平菜单，白色背景
- 选中项使用小米橙色标识
- 菜单文字颜色调整为深灰色
- 悬停效果优化

### 2.3 Card组件
- 白色背景，8px圆角
- 轻微阴影效果
- 适当内边距和外边距
- 标题文字加粗

### 2.4 Button组件
- 主要按钮使用小米橙色
- 次要按钮使用白色背景+橙色边框
- 圆角设计
- 悬停效果优化

### 2.5 Chart组件
- 图表颜色调整为小米橙色系
- 图表背景色透明
- 坐标轴样式优化
- 图例样式调整

## 3. 响应式设计优化

### 3.1 移动端适配 (≤768px)
- 菜单折叠为汉堡菜单
- 卡片布局改为单列
- 减少内边距和外边距
- 优化字体大小
- 按钮尺寸调整

### 3.2 平板适配 (769px-1024px)
- 菜单保持水平布局
- 卡片布局调整为2列
- 适当调整组件大小

### 3.3 桌面适配 (≥1025px)
- 完整布局展示
- 卡片布局根据屏幕宽度调整为3-4列
- 充分利用屏幕空间

## 4. 动画效果添加

### 4.1 页面切换动画
- 淡入淡出效果
- 滑动效果

### 4.2 组件加载动画
- 卡片加载时的渐显效果
- 图表加载时的动画效果

### 4.3 交互反馈动画
- 按钮点击反馈
- 菜单悬停效果
- 卡片悬停效果

## 5. 主题配置

### 5.1 Ant Design主题定制
- 使用Ant Design的主题配置功能
- 统一配置色彩、字体、圆角等

### 5.2 全局样式覆盖
- 使用CSS变量定义主题变量
- 在全局样式中覆盖默认样式
